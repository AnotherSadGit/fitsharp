Configure Fixture
The Configure Fixture lets us issue commands to control different facilities of the Fit framework. The format of a configure command is:
  • configure <facility> <command>
Logging

The logging facility can capture an execution trace of the core fitSharp engine and display it. In its present form it is probably mostly useful to fitSharp framework developers only. We can issue these commands:
  • configure logging start
  • configure logging stop
  • configure logging show
Symbols

We can set, display and clear Symbol Values.

when input isconfigure symbols save mysymbol 'value of System.String' myvalue

configure symbols 'get value' mysymbol

configure symbols clear
