fitSharp
Fit
fitSharp includes a .NET implementation of the Framework for Integrated Test (Fit) tool, including a .NET version of FitLibrary. You can use it to test .NET applications with story tests stored in the FitNesse wiki or in plain old HTML files.

Here's what's available on this site:
  • The Feature List describes what fitSharp Fit can do. This is a work in progress that is being enhanced with each release.
  • The Sample Input Files and Sample Output Files are used by the story tests for fitSharp Fit.
  • Some Unit Tests have been written with Fit to exercise its capabilities.
  • Useful Links will take you to more information about Fit.
