App Config Files And Slim
If your application uses an application configuration file, e.g. myapp.exe.config, it won't be loaded by default when the fitSharp Slim Server runs. To specify the application configuration file, change the COMMAND_PATTERN definition.
!define COMMAND_PATTERN {%m -r fitSharp.Slim.Service.Runner,c:\program files\fitsharp\fitsharp.dll -a c:\mypath\myapp.exe.config %p}
